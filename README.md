本项目基于Ruby on Rails框架建立web应用，实现了以下功能：
　主页链接到国科大官网；
  显示当前时间；
  点击“查看课程”按钮跳转至课程页面；
  在课程页面可实现对数据库增删查改操作，并在前端进行显示。
  
项目代码托管到Cloud９云服务上，地址：https://course-tianleh.c9users.io/
